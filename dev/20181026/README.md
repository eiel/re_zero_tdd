# FizzBuzz

## 仕様

- ３で割り切れる場合は Fizz を返す
- 5 で割り切れる場合は Buzz を返す
- 両方で割り切れる場合は FizzBuzz を返す
- それ以外では値を返す
- 繰り返し実行できるようにする

## 設計

### TODO リスト
